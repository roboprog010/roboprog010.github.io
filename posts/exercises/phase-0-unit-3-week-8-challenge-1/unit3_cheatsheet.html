JavaScript Object Cheatsheet

Literal Syntax

Create an Object

var ship = {};

Create an Object with Properties and Methods

var ship = { className: 'fighter', fireMissle: function() { // code to fire missle } };

Add Properties

ship.className = 'fighter';

Add Methods

ship.fireMissle = function() { // code to fire missle };

Accessing Properties and Methods

Properties

ship.className; ship['className'];

Methods

ship.fireMissle();

Removing Properties

delete ship.className;

Object Create Method

var FighterShip = { className: 'fighter', fireMissle: function() { // code to fire missle } }; var ship_x1 = Object.create(FighterShip);

Constructor Function

function Ship (className) { this.className = className; this.fireMissle = function() { // code to fire missle }; } ship = new Ship('fighter');

Looping

for (var key in ship) { console.log(key + ' ' + ship[key]); }

Prototyping

Ship.prototype.missleCount = 10; Ship.prototype.missleDisplay = function() { // code to display missle count };

Comparing Objects

Object variables are reference types and therefore cannot be compared using a standard test of equality. A custom function or method must be created to iterate over each property of two objects and compare their values one to one.
